# Curso Alura - Spring Data JPA: Repositórios, Consultas, Projeções e Specifications

## Desafio do Projeto
- Utilizar o Spring Data para integrar com o banco de dados;
- Configurar o ambiente da nossa aplicação para conectar com o Framework;
- Criar e representar entidades que vão ser criadas também no banco de dados de forma dinâmica;
- Fazer consultas com todas as possibilidades que o Framework disponibiliza;
- Criar paginação e ordenação com o Framework;
- Criar querys projetadas retornando apenas os dados que queremos apresentar;
- Por fim, ver todo o dinamismo que o Framework nos dá para fazer querys dinâmicas.

## Stack

- Java 11
- Spring Data JPA
- MariaDB
